Práctica Final — Repaso de JavaScript

Realizado por Jose Izquierdo

Esta práctica resume los conceptos fundamentales de JavaScript mediante ejemplos y ejercicios interactivos.

Variables y tipos

Las variables permiten almacenar datos. Pueden ser números, textos, booleanos u otros tipos.

let edad = 20;
const nombre = "Ana";
let activo = true;

Operadores

Los operadores permiten realizar cálculos, comparaciones y operaciones lógicas.

let x = 10;
let y = 3;
console.log(x + y);
console.log(x > y);

Condicionales

Los condicionales permiten ejecutar código solo cuando se cumple una condición.

let nota = 6;
if (nota >= 5) console.log("Aprobado");

Bucles

Los bucles permiten repetir instrucciones varias veces de forma automática.

for (let i = 1; i <= 5; i++) {
  console.log(i);
}

    Arrays

    Un array permite almacenar varios valores en una sola variable.

    let frutas = ["manzana", "plátano", "naranja"];
console.log(frutas[0]);

    Funciones

    Las funciones permiten agrupar código reutilizable que se puede ejecutar cuando se necesite.

    function sumar(a, b) {
  return a + b;
}

    DOM

    El DOM permite acceder y modificar elementos HTML desde JavaScript.

    document.getElementById("titulo").textContent = "Nuevo texto";

    Este texto se puede cambiar.

    Eventos

    Los eventos permiten ejecutar código cuando el usuario interactúa con la página, como por ejemplo al hacer clic

    elemento.addEventListener("click", () => {});

    Math

    El objeto Math ofrece funciones matemáticas como redondeos y generación de números aleatorios.

    Math.random();
Math.floor();

    Strings

    Los strings representan texto y permiten realizar operaciones como mayúsculas, búsqueda o reemplazo.

    let nombre = "María";
console.log(nombre.toUpperCase());
